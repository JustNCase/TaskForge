"### TaskForge Project Scripts"  
""  
"This directory contains scripts and documentation for the TaskForge project."  
""  
"## Available Scripts"  
""  
"| Script | Description |"  
"|--------|-------------|"  
"| \`typecheck\` | Run TypeScript type checking to validate type definitions |"  
"| \`lint\` | Run linting to catch code style and potential issues |"  
"| \`tests\` | Run all tests for the project |"  
"| \`docs\` | Generate documentation |"  
"| \`setup\` | Setup environment and install dependencies |"  
""  
"## Project Structure"  
""  
"This repository contains:"  
"- \`apps/\` - The application code (Web, API, Dashboard)"  
"- \`packages/\` - Shared libraries and packages"  
"- \`Components/\` - Shared React components"  
"- \`database/\` - Database schemas and migrations"  
"- \`docs/\` - Project documentation"  
"- \`scripts/\` - Utility scripts and automation scripts"  
""  
"## Project Purpose"  
""  
"TaskForge is a comprehensive task management and productivity application that leverages AI to help users organize, prioritize, and complete their tasks effectively."
